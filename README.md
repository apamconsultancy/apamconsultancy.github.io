# APaM Consultancy Website - Summary

## Overview
A clean, minimal two-page website for an NGO consultancy firm, featuring modern design with minimalist SVG icons and a professional color scheme based on the company logo.

## Pages Created

### 1. Home Page (index.html)
- **Hero Section**: Eye-catching introduction with gradient background
- **Services Section**: 6 core services with SVG icons
  - M&E System Set-up
  - Data Management Set-up
  - Survey and Assessment
  - Evaluation Services
  - Tailored Training
  - Consultancy Linkage
- **Approach Section**: 3 service delivery models
- **Target Clients Section**: Primary and secondary partners
- **CTA Section**: Call-to-action for engagement
- **Contact Section**: Email, coverage area, and partnership info

### 2. About Us Page (about.html)
- **Vision Section**: Company vision statement
- **Mission Section**: Core mission and approach
- **Core Values**: 4 key values with SVG icons
  - Integrity
  - Partnership
  - Innovation
  - Adaptability
- **Organizational Structure**: 4 team components
  - Founders & Core Members
  - Expert Consultants
  - Dedicated Volunteers
  - Field Enumerators
- **Governance Section**: Project management approach
- **Team Section**: Core team members
- **Code of Conduct**: 5 principles with SVG icons

## Design Features

### Color Scheme (from logo)
- Primary Teal: #1a9b8e
- Primary Coral: #ff6b4a
- Accent Orange: #ff8c42
- Dark Teal: #147a6f
- Light Teal: #e8f5f3

### Typography
- Headings: Outfit (Google Fonts)
- Body: Inter (Google Fonts)

### Icons
- All icons are minimalist SVG graphics
- Consistent stroke-based design
- Teal color scheme matching brand
- Hover animations and transitions

### Interactive Elements
- Smooth scrolling navigation
- Mobile-responsive hamburger menu
- Card hover effects with elevation
- Fade-in animations on scroll
- Button ripple effects
